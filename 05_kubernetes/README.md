# Container Orchestration with Kubernetes (K8s)

Managing containerized workloads at scale using Kubernetes clusters.

### Learning Path:
* **Pod & Deployment Management:** Handling scaling and self-healing.
* **Services & Ingress:** Managing networking and external access.
* **ConfigMaps & Secrets:** Decoupling configuration from application code.
* **Helm:** Packaging and deploying complex K8s applications.
